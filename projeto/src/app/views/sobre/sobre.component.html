Este é um projeto de Front-End, Back-End e banco de dados



Possui tema livre, desde que atenda aos critérios mínimos, listados a seguir:

    Definição Geral – Serviços back end

  • -> 3 CRUDs (funcionalidades básicas de cadastro, alteração, deleção, listagem e pesquisa)
  • -> Deve possuir pelo menos uma funcionalidade de negócio (por exemplo, venda, reserva, etc)
  • -> Deve possuir pelo menos uma funcionalidade de relatório (por exemplo, relatório de vendas mensais ou relatório de produtos mais vendidos no mês)
  • -> As entidades devem estar completas, ou seja, devem possuir todos os atributos e validações necessários para que funcionem corretamente e agreguem algum valor ao uso do sistema
  • -> O projeto deve ser feito utilizando tecnologias vistas na disciplina e no semestre
  • -> O projeto deve ter integração completa com banco de dados MySQL
